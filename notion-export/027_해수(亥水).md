# 해수(亥水)

Source: https://www.notion.so/1d96a739fb85806ba6a6ce7fc8f3b494

---

### 1. 기본 속성 요약
| 항목 | 내용 |
| --- | --- |
| 한자 표기 | 亥 (해수) |
| 위치 | 지지(地支) – 12지지 중 열두 번째 |
| 오행 속성 | 수(水) – 생지(水) |
| 음양 속성 | 양(陽) |
| 계절 | 겨울의 시작 (음력 10월 ~ 11월) |
| 시간대 | 오전 9시 ~ 11시 |
| 에너지 키워드 | 준비, 축적, 통찰, 변화의 씨앗, 재생 |
| 상징 이미지 | 겨울의 씨앗, 동면, 지하수, 동지 무렵의 고요함 |

---

### 2. 해수의 주요 특성
| 분류 | 설명 |
| --- | --- |
| 시기적 상징 | 한 해의 마무리이자 다음 순환의 준비 시기 |
| 에너지 성질 | 음기 절정 속 양기 싹틈 – 정리와 시작이 공존 |
| 내면 의미 | 깊은 성찰과 계획, 미래에 대한 준비 |
| 핵심 키워드 | 내면화, 준비, 침잠, 통찰, 근본적 변화의 시작 |

---

### 3. 해수의 이미지와 상징
| 분류 | 상징 |
| --- | --- |
| 자연물 | 지하수, 씨앗, 고요한 바다, 눈 내린 들판 |
| 감각 키워드 | 고요, 깊음, 냉정, 준비, 숨은 성장 |
| 계절감 | 겨울의 시작, 음기 절정기 |
| 성질 | 사려 깊음, 체계적 준비, 내면적 통찰 |
| 추상 개념 | 회복력, 변화관리, 전략기획, 장기계획 |

---

### 4. 해수의 지장간 구성
| 지장간 | 의미 | 성향 | 역할 설명 |
| --- | --- | --- | --- |
| 무토(戊土) | 정리와 안정 | 보관, 질서 | 과거를 정리하고 내면 정비 |
| 갑목(甲木) | 시작과 목표 설정 | 진취적, 계획적 | 봄을 향한 방향성과 의지 |
| 임수(壬水) | 실행 준비와 통찰 | 신중함, 예측력 | 변화 대비 및 전략적 준비 |

---

### 5. 현대적 해석과 적용
| 키워드 | 설명 |
| --- | --- |
| 적응력 | 변화의 시작을 받아들이고 준비하는 유연한 사고 |
| 혁신성 | 새롭게 떠오르는 가능성과 아이디어의 씨앗 |
| 회복력 | 고요함 속에서 미래를 준비하며 재정비 |
| 역량 키워드 | 설명 |
| --- | --- |
| 전략 기획력 | 큰 그림을 보고 장기 계획을 수립할 수 있음 |
| 준비력 | 철저한 분석과 실행 준비 능력 |
| 변화관리 역량 | 변화의 조짐을 빠르게 포착하고 대응하는 능력 |

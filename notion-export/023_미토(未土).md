### 2. 미토의 주요 특성
| 분류 | 설명 |
| --- | --- |
| 시기적 상징 | 여름과 가을 사이의 전환점 – 수확 준비 시기 |
| 에너지 성질 | 양기의 감소와 음기의 시작 – 안정, 정리, 결실 중심 |
| 내면 의미 | 그간의 경험을 정리하고, 내실 있는 변화 준비 |
| 핵심 키워드 | 통합, 정돈, 실천, 전략, 실용성 |

### 3. 미토의 이미지와 상징
| 분류 | 상징 |
| --- | --- |
| 자연물 | 성숙한 곡물, 수확 직전의 들판, 건조한 대지 |
| 감각 키워드 | 무르익음, 쌓임, 익어감 |
| 계절감 | 여름 말기의 열기와 가을 전환의 서늘함 공존 |
| 성질 | 실용, 축적, 정리, 계획 중심 |
| 추상 개념 | 수렴, 안정화, 결산, 변화 준비 |

### 4. 미토의 지장간 구성
| 지장간 | 의미 | 성향 | 역할 |
| --- | --- | --- | --- |
| 정화(丁火) | 여름의 남은 열기 | 실천·행동 중심 | 실질적 결과와 구체적 성과 도출 |
| 을목(乙木) | 지나온 성장의 기록 | 경험·기억 | 사회 현상과 지식의 정리, 이론-실용 연결 |
| 기토(己土) | 대지 그 자체 | 보존·수렴 | 경험을 실용으로 정리하고 향후 기반 마련 |

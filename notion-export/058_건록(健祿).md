## 🏛 건록(建祿) – 성숙한 자립성과 현실적 성취의 운성

---

### 🔍 기본 개념
- 건록은 십이운성 중 관대(冠帶) 다음 단계로,사회적 기반을 확립하고 자립적 성공을 이뤄가는 성숙한 장년기의 운성입니다.
- 한자 의미:세울 건(建) + 복록 록(祿) → 스스로 부와 지위를 세운다
- 전통 명리학에서는 현실성, 추진력, 자수성가의 상징으로 높게 평가됨

---

### 😇 건록의 상징적 의미
| 특성 키워드 | 설명 |
| --- | --- |
| 자립성 · 독립심 | 부모·타인에게 기대지 않고 스스로 일어서는 힘 |
| 고집 · 추진력 | 확고한 방향성과 실행력, 때론 아집으로 작용 |
| 강한 컨디션 | 육체적·정신적으로 성숙된 최정점의 시기 |
| 직장형 기질 | 안정성 중시, 조직 속에서 성장하는 사주 |
| 인간관계 우호 | 책임감 있게 주변 챙기며 신뢰 쌓는 스타일 |

---

### ☀️ 건록의 특징 요약
- 💼 사회 기반 위에서 자수성가 가능
- 🚀 추진력과 실행력 중심의 리더형 에너지
- 🧠 전략·계획에 강하지만 독단적 성향 있음
- 👥 친구·형제와 정이 깊고, 협력도 잘함
- 💸 재물복은 있으나 소비 습관 조절 필요

## 🧠 건록(建祿)의 성격 특성

건록은 십이운성 중에서도 가장 강한 현실력과 자기주도성을 지닌 운성입니다.

그 기질은 안정적 성취, 리더십, 독립적 생존력으로 나타납니다.

---

### ✅ 긍정 성격 특성
| 성격 특성 | 설명 |
| --- | --- |
| 실행력 · 성취력 | 추진력 뛰어나고 실행 기반의 실적형 |
| 자수성가 | 스스로 성공 구조를 만드는 독립적 힘 |
| 리더십 | 조직과 팀을 이끄는 중간 관리자형 |
| 전략적 사고 | 계획적이며 문제 해결에 실리적으로 접근 |
| 책임감 | 자기 일에 대한 통제력과 책임의식 우수 |

---

### ⚠️ 보완이 필요한 성격 특성
| 성격 특성 | 설명 |
| --- | --- |
| 고집과 독단 | 타인의 조언을 수용하지 않으려는 태도 |
| 배타적 태도 | 내 방식만 옳다고 생각, 융통성 부족 |
| 감정 표현 미숙 | 정은 깊으나 표현에 서툴 수 있음 |
| 현실 낙관 | 리스크 경시, 지나친 기대 → 손실 가능성 |
| 소비 과다 | 친구·지인에게 잘 쓰지만 정작 자신은 저축 부족 |

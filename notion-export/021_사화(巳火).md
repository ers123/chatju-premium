### 1. 기본 속성 요약

| --- | --- |

| 성향 요약 | 열정적, 사교적, 추진력과 실행력을 겸비한 성취형 에너지 |

### 2. 사화의 주요 특성
| 분류 | 설명 |
| --- | --- |
| 시기적 상징 | 봄의 열매가 드러나며 여름을 여는 문턱 |
| 에너지 성질 | 양기(陽氣)의 정점, 외부로의 활발한 표출 |
| 내면 의미 | 열정과 사회적 활동을 통해 성취를 이루려는 에너지 |
| 핵심 키워드 | 실행력, 네트워킹, 성취, 외향성, 리더십 |

### 3. 사화의 이미지와 상징
| 분류 | 상징 |
| --- | --- |
| 자연물 | 뜨거운 태양, 피어난 꽃, 불꽃 |
| 동물상 | 뱀 – 유연하면서도 강한 생명력 |
| 계절감 | 여름의 시작 – 활동과 성취의 시기 |
| 성질 | 사교적, 열정적, 활달함 |
| 추상 개념 | 실행, 성취, 관계, 활동, 확장 |

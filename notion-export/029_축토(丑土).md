### 1. 기본 속성 요약
| 항목 | 내용 |
| --- | --- |
| 한자 표기 | 丑 (축토) |
| 위치 | 지지(地支) – 12지지 중 두 번째 |
| 오행 속성 | 토(土) – 고지(庫地) |
| 음양 속성 | 음(陰) |
| 계절 | 겨울의 말미 (음력 12월) |
| 시간대 | 축시(丑時) – 새벽 1시 ~ 3시 |
| 에너지 키워드 | 저장, 축적, 응축, 정제, 전환 |
| 상징 이미지 | 창고, 땅속 저장소, 숙성 창고, 정리된 도서관 |

### 2. 축토의 핵심 특성
| 분류 | 설명 |
| --- | --- |
| 시기적 상징 | 겨울의 끝, 봄을 준비하는 저장 단계 |
| 에너지 성질 | 정적인 토(土) 기운, 깊이 응축된 에너지 |
| 내면 의미 | 축적과 정리, 성숙한 지혜의 보존 |
| 핵심 키워드 | 고요, 보관, 숙성, 실천 준비, 다음 단계 이행 |

### 3. 축토의 이미지와 상징
| 분류 | 상징 |
| --- | --- |
| 자연물 | 땅속 창고, 발효 저장고, 겨울의 대지 |

| 성질 | 정제된 판단력, 고도의 분석력, 실용 중심 사고 |

### 4. 축토의 지장간 구성
| 지장간 | 의미 | 성향 | 역할 설명 |
| --- | --- | --- | --- |
| 계수(癸水) | 보관, 유동성 | 차분하고 내향적 | 축적된 지식을 안정적으로 보존하고 이어주는 역할 |
| 신금(辛金) | 정제, 숙성 | 섬세하고 완결지향적 | 경험을 다듬고 고품질로 정제하여 다음 단계로 전환 |
| 기토(己土) | 응축, 조정 | 중심축, 조율자 | 응축된 자원을 체계화하여 전달 가능 상태로 전환 |

### 5. 현대적 해석과 적용
| 키워드 | 설명 |
| --- | --- |
| 적응력 | 과거의 경험을 체계적으로 정리해 미래 변화에 유연 대응 |
| 축적성 | 지식, 데이터, 경험을 안정적으로 쌓고 유지 |
| 안정성 | 계획과 실천을 위한 믿음직한 기반 제공 |
| 역량 키워드 | 설명 |
| --- | --- |
| 전문성 개발 | 깊이 있는 분야 지식 구축 및 유지에 강함 |
| 지식 관리 능력 | 조직 내 전수, 후속자 교육, 매뉴얼화 등 강점 |
| 전략적 실행력 | 계획을 구체화하고 실천으로 연결하는 실용적 사고 |

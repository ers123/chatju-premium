### 1. 기본 속성 요약
| 항목 | 내용 |
| --- | --- |
| 한자 표기 | 申 (신금) |
| 지지 위치 | 지지(地支) – 12지지 중 아홉 번째 |
| 오행 속성 | 금(金) |
| 음양 속성 | 양(陽) |
| 계절 | 가을의 시작 (음력 7월) |
| 시간대 | 오후 3시 ~ 5시 |
| 에너지 키워드 | 정제, 통찰, 결실, 전략 |
| 대표 이미지 | 맑은 가을 하늘, 정련된 금속, 수확 준비 |
| 성향 요약 | 분석적, 전략적, 철저함, 정리정돈형 |

---

### 2. 신금의 주요 특성
| 분류 | 설명 |
| --- | --- |
| 시기적 상징 | 여름이 끝나고 가을이 시작되는 시기 – 결실을 준비함 |
| 에너지 성질 | 양금의 견고함과 냉철함, 결과 중심의 실질적 성향 |
| 내면 의미 | 성과를 정리하고, 핵심만 남기는 ‘정제의 시기’ |
| 핵심 키워드 | 검증, 본질, 효율, 수렴, 실력 중심 |

---

### 3. 신금의 이미지와 상징
| 분류 | 상징 |
| --- | --- |
| 자연물 | 금속 광석, 칼날, 수확 도구 |
| 감각 키워드 | 단단함, 예리함, 정리된 공간 |
| 계절감 | 초가을의 맑고 선선한 기운 |
| 성질 | 분석적, 치밀함, 판단력 우위 |
| 추상 개념 | 통찰, 전략, 실적, 정제, 룰 |

---

### 4. 신금의 지장간 구성
| 지장간 | 의미 | 성향 | 역할 |
| --- | --- | --- | --- |
| 무토(戊土) | 기반 · 구조 | 안정성과 수용력 | 정제와 실행을 위한 내적 준비 |
| 임수(壬水) | 통찰 · 계획 | 미래 지향적 사고 | 목표 설정과 전략 수립 |
| 경금(庚金) | 실력 · 성과 | 단단함과 검증 | 실질적 성과 창출 및 실행력 발휘 |

---

### 5. 현대적 해석과 적용
| 특성 | 설명 |
| --- | --- |
| 적응력 | 불필요한 것을 제거하고 본질을 남기는 정리 능력 |
| 혁신성 | 분석을 통한 구조 재편과 실질적 성과 기반 혁신 |
| 실행력 | 세운 계획을 체계적으로 실현해내는 실행능력 |
| 역량 | 설명 |
| --- | --- |
| 분석력 | 문제의 핵심을 파악하고 전략화 |
| 실행력 | 실무 추진력과 성과 중심 마인드 |
| 전략적 사고 | 정제된 판단을 바탕으로 장기 플랜 수립 |

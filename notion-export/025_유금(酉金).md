### 1. 기본 속성 요약
| 항목 | 내용 |
| --- | --- |
| 한자 표기 | 酉 (유금) |
| 지지 위치 | 지지(地支) – 12지지 중 여덟 번째 |
| 오행 속성 | 금(金) |
| 음양 속성 | 음(陰) |
| 계절 | 가을의 절정 (음력 8월) |
| 시간대 | 오후 5시 ~ 7시 |
| 에너지 키워드 | 완성, 수확, 실용성, 구조화 |
| 대표 이미지 | 황금빛 들녘, 수확하는 손길, 정제된 금속 |
| 성향 요약 | 결과지향, 디테일중시, 책임감, 정교함 |

---

### 2. 유금의 주요 특성
| 분류 | 설명 |
| --- | --- |
| 시기적 상징 | 가을의 절정기 – 결실과 성숙, 완성의 시기 |
| 에너지 성질 | 음금의 정제된 기운 – 효율, 실용, 실력 강조 |
| 내면 의미 | 무르익은 결과물의 정리·정돈과 전달 |
| 핵심 키워드 | 완성도, 실용성, 전문성, 전략적 판단력 |

---

### 3. 유금의 이미지와 상징
| 분류 | 상징 |
| --- | --- |
| 자연물 | 금속 공예품, 정제된 칼날, 포장된 상품 |
| 감각 키워드 | 정제, 완성, 매끄러움, 깔끔함 |
| 계절감 | 맑고 차가운 가을 저녁의 기운 |
| 성질 | 섬세함, 정리력, 집중력 |
| 추상 개념 | 완성도, 실천력, 성과 중심적 태도 |

---

### 4. 유금의 지장간 구성
| 지장간 | 의미 | 성향 | 역할 |
| --- | --- | --- | --- |
| 경금(庚金) | 생산, 결과 도출 | 추진력, 거침 | 실무 실행 및 성과 창출 |
| 신금(辛金) | 정제, 디테일 | 정교함, 소비자 시각 | 제품의 완성도 및 전달 준비 |

---

### 5. 현대적 해석과 적용
| 특성 | 설명 |
| --- | --- |
| 적응력 | 결과 중심의 실용적 사고로 빠른 시장 적응 |
| 혁신성 | 기존 틀을 다듬고 개선하는 완성형 혁신 |
| 회복력 | 위기 후 빠르게 정비하고 개선하는 체계적 복구력 |
| 역량 | 설명 |
| --- | --- |
| 실행력 | 프로젝트 마무리 능력, 실질적 성과 창출 |
| 정제력 | 완성도 높은 작업물 추구, 소비자 만족도 고려 |
| 전략성 | 결실 시기를 바라보는 효율적 기획 역량 |

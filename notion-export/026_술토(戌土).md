### 1. 기본 속성 요약
| 항목 | 내용 |
| --- | --- |
| 한자 표기 | 戌 (술토) |
| 위치 | 지지(地支) – 12지지 중 아홉 번째 |
| 오행 속성 | 토(土) – 고지(土) |
| 음양 속성 | 양(陽) |

| 시간대 | 오후 7시 ~ 9시 |
| 에너지 키워드 | 정리, 저장, 완성, 균형, 응집 |
| 상징 이미지 | 수확 창고, 정리된 서고, 응축된 에너지 |
| 성향 요약 | 신중함, 체계성, 마무리 능력, 내실 중심 |

---

### 2. 술토의 주요 특성
| 분류 | 설명 |
| --- | --- |
| 시기적 상징 | 가을의 마무리 – 수확을 마치고 저장하는 시기 |
| 에너지 성질 | 응축, 정리, 조정, 분류의 기운이 강함 |
| 내면 의미 | 결과물의 체계화 및 후속 작업을 위한 준비 |
| 핵심 키워드 | 분별력, 안정감, 정리력, 체계적인 축적 |

---

### 3. 술토의 이미지와 상징
| 분류 | 상징 |
| --- | --- |
| 자연물 | 낙엽 쌓인 들판, 정리된 곡식 창고 |
| 감각 키워드 | 응축, 마감, 결실, 숙성 |
| 계절감 | 쓸쓸하지만 풍요로운 늦가을, 수확의 정점 |
| 성질 | 안정감, 포용성, 마무리형 조직력 |
| 추상 개념 | 마무리, 정리, 조정, 정제, 준비 |

---

### 4. 술토의 지장간 구성
| 지장간 | 의미 | 성향 | 역할 설명 |
| --- | --- | --- | --- |
| 신금(辛金) | 정제, 세밀함 | 분석적, 수정력 | 수요자 맞춤형 정리 및 품질 보완 |
| 정화(丁火) | 활용, 응용력 | 섬세함, 조율력 | 결과물의 사용자 적용성 강화 |
| 무토(戊土) | 기반, 안정성 | 구조화, 내구성 | 결과물의 보존 및 다음 단계를 위한 조정 |

### 5. 현대적 해석과 적용
| 키워드 | 설명 |
| --- | --- |
| 적응력 | 변화 속에서도 구조를 유지하는 안정형 대응력 |
| 혁신성 | 경험 축적을 바탕으로 본질적인 개선 및 전략 수립 |
| 회복력 | 위기를 정리하고 다시 체계를 회복하는 내구성 |
| 역량 키워드 | 설명 |
| --- | --- |
| 정리력 | 복잡한 정보나 업무를 체계적으로 정돈 |
| 마무리 실행력 | 프로젝트를 끝까지 책임지고 완결짓는 능력 |
| 전략적 판단력 | 쌓인 데이터를 기반으로 미래를 준비하고 설계 |

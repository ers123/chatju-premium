# 자수(子水)

### 1. 기본 속성 요약
| 항목 | 내용 |
| --- | --- |
| 한자 표기 | 子 (자수) |
| 위치 | 지지(地支) – 12지지 중 첫 번째 |
| 오행 속성 | 수(水) – 왕지(水) |
| 음양 속성 | 음(陰) |
| 계절 | 겨울 한복판 (음력 11월) |
| 시간대 | 자시(子時) – 자정(밤 11시~1시) |
| 에너지 키워드 | 시작, 도전, 내면의 힘, 방향성, 실행 |
| 상징 이미지 | 북쪽의 깊은 바다, 씨앗의 발아, 새벽의 정적 |

---

### 2. 자수의 핵심 특성
| 분류 | 설명 |
| --- | --- |
| 시기적 상징 | 음이 극대화된 가운데 양기가 싹트는 전환점 |
| 에너지 성질 | 겨울 중심의 수기(水氣)가 가장 왕성하게 발현됨 |
| 내면 의미 | 고요함 속에 숨은 강한 추진력과 도전 정신 |
| 핵심 키워드 | 깊이, 주도성, 통찰, 실행력, 새로운 방향성 |

---

### 3. 자수의 이미지와 상징
| 분류 | 상징 |
| --- | --- |
| 자연물 | 북쪽의 깊은 물, 겨울 바다, 씨앗 속 싹 |
| 감각 키워드 | 차분함, 깊이, 정적, 통찰 |
| 성질 | 실행 중심, 도전적, 체계적 사고, 원칙 중시 |
| 추상 개념 | 방향 설정, 변화 촉진, 내면 성장, 확장성과 실행력 |

---

### 4. 자수의 지장간 구성
| 지장간 | 의미 | 성향 | 역할 설명 |
| --- | --- | --- | --- |
| 임수(壬水) | 겨울의 확장력 | 외향적, 도전적 | 본격적인 활동과 외부 확장의 동력 |
| 계수(癸水) | 수렴과 내면화 | 내향적, 세밀한 | 임수의 과잉 확장을 조율, 내실 강화 |

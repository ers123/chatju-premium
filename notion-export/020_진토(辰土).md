# 진토(辰土)

### 1. 기본 속성 요약
| 항목 | 내용 |
| --- | --- |
| 한자 표기 | 辰 (진토) |
| 지지 위치 | 지지(地支) – 12지지 중 다섯 번째 |
| 오행 속성 | 토(土) |
| 음양 속성 | 양(陽) |
| 계절 | 봄의 마지막 (음력 3월) |
| 방위 | 동남쪽 (동쪽의 끝자락, 남쪽의 시작) |
| 시간대 | 오전 7~9시 (빛이 퍼지고 활동이 시작되는 시점) |
| 에너지 키워드 | 전환, 축적, 시작의 마무리, 실행의 준비 |
| 대표 이미지 | 생명이 깨어나는 대지, 묵직한 변화의 물결 |
| 성향 요약 | 다재다능, 실용성, 축적된 경험을 바탕으로 창의적 실행 가능 |

---

### 2. 진토의 주요 특성
| 분류 | 설명 |
| --- | --- |
| 시기적 상징 | 봄의 마지막 – 다음 계절을 준비하는 시점 |
| 에너지 성질 | 축적된 에너지의 정리, 정적인 것에서 활동으로 전환 |
| 내면 의미 | 과거의 경험이 새로운 성과로 이어지는 전환기 |
| 핵심 키워드 | 경험, 실행, 적응, 회복, 창조적 실용성 |

---

### 3. 진토의 이미지와 상징
| 분류 | 상징 |
| --- | --- |
| 자연물 | 대지, 창고, 새싹이 움트는 땅 |
| 방향성 | 변화의 문턱, 겨울에서 봄을 지나 여름으로 가는 지점 |
| 계절감 | 고지(庫地) – 에너지와 지혜가 축적된 저장고 같은 땅 |
| 성질 | 실용성, 적응력, 전환의 유연함 |
| 추상 개념 | 변화의 다리, 정리와 실행, 현실화되는 가능성 |

---

### 4. 진토의 지장간 구성
| 지장간 | 상징 | 역할 |
| --- | --- | --- |
| 을목(乙木) | 봄의 성장 | 경험을 체계화하여 실행 지침으로 만들기 |
| 계수(癸水) | 깊은 통찰 | 사람과 관계에 대한 이해력 증진 |
| 무토(戊土) | 실용과 결실 | 경험을 실제 성과로 전환하고 실용화함 |

---

### 5. 현대적 해석과 적용
| 역량 | 설명 |
| --- | --- |
| 적응력 | 변화하는 환경에 유연하게 대처 |
| 창의성 | 축적된 경험을 바탕으로 창의적 해결책 제시 |
| 실용성 | 현실적 실행력과 문제해결 중심 사고 |
| 회복력 | 어려운 상황에서 다시 일어나는 힘 |
| 전략적 사고 | 경험을 분석하여 장기적 계획 수립 가능 |

### 1. 기본 속성 요약
| 항목 | 내용 |
| --- | --- |
| 한자 표기 | 午 (오화) |
| 지지 위치 | 지지(地支) – 12지지 중 일곱 번째 |
| 오행 속성 | 화(火) |
| 음양 속성 | 음(陰) |
| 계절 | 여름의 절정 (음력 5월) |
| 시간대 | 오전 11시 ~ 13시 (태양이 가장 높이 뜨는 시간) |
| 에너지 키워드 | 열정, 성취, 주도성, 표현력 |
| 대표 이미지 | 정오의 태양, 활활 타오르는 불꽃 |
| 성향 요약 | 강한 추진력, 대외적 활동성, 성과 중심, 주도적인 에너지 |

---

### 2. 오화의 주요 특성
| 분류 | 설명 |
| --- | --- |
| 시기적 상징 | 하지 근처, 모든 에너지가 극대화된 시기 |
| 에너지 성질 | 열정과 활력이 최고조, 결과 중심 사고 |
| 내면 의미 | 성취 욕구, 사회적 영향력, 리더십 강화 |
| 핵심 키워드 | 실행력, 추진력, 성과 지향, 관계 확장 |

---

### 3. 오화의 이미지와 상징
| 분류 | 상징 |
| --- | --- |
| 자연물 | 정오의 태양, 불길, 화로 |
| 동물상 | 말 – 활동성, 속도감, 방향성 |
| 계절감 | 여름의 절정 – 결실 전 단계의 에너지 폭발 |
| 성질 | 외향적, 성취지향, 주도적, 다혈질 |
| 추상 개념 | 추진, 확장, 영향력, 성과 |

---

### 4. 오화의 지장간 구성
| 지장간 | 의미 | 성질 | 역할 |
| --- | --- | --- | --- |
| 병화(丙火) | 여름의 대표 화기 | 외향적·활동적 | 리더십, 사회적 네트워크 형성 |
| 정화(丁火) | 내면의 성숙한 화기 | 섬세·정제 | 실제적 결과 창출, 성과 수확 |

---

### 5. 현대적 해석과 적용
| 특성 | 설명 |
| --- | --- |
| 적응력 | 빠르게 상황 판단 후 실천하는 유연한 리더십 |
| 활력성 | 팀 내 에너지 메이커, 위기에서 돌파구를 만드는 힘 |
| 표현력 | 자기 주장과 아이디어를 적극적으로 전달함 |
| 역량 | 설명 |
| --- | --- |
| 추진력 | 아이디어를 실행에 옮기고 끝까지 밀어붙이는 힘 |
| 리더십 | 팀과 조직을 주도하며, 중심을 잡는 역량 |
| 성과 창출 | 성취 지향적 사고로 목표 달성에 집중 |
